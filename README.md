# 100days-apps

AIを活用しながら100日間アプリ開発に挑戦するリポジトリです。

## Day1: じゃんけんゲーム

HTML / CSS / JavaScriptで作成。

### できること
- グー・チョキ・パーを選べる
- CPUがランダムに手を出す
- あいこ・勝ちの判定

## Day2: 電卓アプリ

HTML / CSS / JavaScriptで作成。

### できること
- 数字ボタンを押すと表示される
- 四則演算ができる
- Cボタンでリセットできる

## Day3: ToDoアプリ

HTML / CSS / JavaScriptで作成。

### できること
- 入力欄にToDoを入力できる
- 追加ボタンでリストに表示できる
- 空欄のまま追加できない
- ❌ボタンで削除できる


## Day4: タイマーアプリ

HTML / CSS / JavaScriptで作成。

### できること
- 25分タイマーを表示できる
- スタートでカウントダウンできる
- ストップで一時停止できる
- リセットで25:00に戻せる
- スタート連打で速度が上がるバグを防止
- 0秒になったら停止して通知する

## Day5: 文字数カウンター

HTML / CSS / JavaScriptで作成。

### できること
- 入力した文章の文字数をリアルタイムで表示できる
- 空白なしの文字数を表示できる
- X投稿用に残り文字数を表示できる

### 学んだこと
- `addEventListener("input")` で入力中に処理を実行する
- `length` で文字数を数える
- `replace(/\s/g, "")` で空白を除外する

## Day7: 健康目安チェッカー

HTML / CSS / JavaScriptで作成。

### できること
- 身長・体重・目標体重を入力できる
- BMIを計算できる
- BMIに応じて判定を表示できる
- 標準体重を表示できる
- 標準体重との差を表示できる
- 目標体重との差を表示できる
- アドバイスを表示できる
- 入力値と体重記録を保存できる

### 学んだこと
- `toFixed(1)` で小数点第1位まで表示する
- `if / else if / else` で判定を分岐する
- `localStorage` でブラウザにデータを保存する


## Day8: ランニングペース計算アプリ

HTML / CSS / JavaScriptで作成。

### できること
- 距離と時間を入力できる
- 1kmあたりのペースを計算できる
- 時速を表示できる
- ペースに応じたアドバイスを表示できる
- 1kmごとの通過タイムを表示できる

### 学んだこと
- `for` 文で繰り返し処理を行う
- 計算結果から一覧を自動生成する
- `padStart()` で秒数を2桁表示にする

## Day9: 営業メールAIプロンプト生成ツール

HTML / CSS / JavaScriptで作成。

### できること
- 営業メール作成用のAIプロンプトを生成できる
- メールの目的を選択できる
- 商談フェーズや相手の温度感を指定できる
- 先方からの返信・商談メモを貼り付けできる
- 懸念点を複数選択できる
- 生成したプロンプトをコピーできる

### 学んだこと
- `select` で選択式の入力を作る
- `checkbox` で複数選択を扱う
- 複数の入力内容を組み合わせて文章を生成する
- `navigator.clipboard.writeText()` でコピー機能を実装する
