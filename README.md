# 100days-apps

AIを活用しながら100日間アプリ開発に挑戦するリポジトリです。

## Day1: じゃんけんゲーム

HTML / CSS / JavaScriptで作成。

### できること
- グー・チョキ・パーを選べる
- CPUがランダムに手を出す
- あいこ・勝ちの判定

## Day2: 電卓アプリ

HTML / CSS / JavaScriptで作成。

### できること
- 数字ボタンを押すと表示される
- 四則演算ができる
- Cボタンでリセットできる

## Day3: ToDoアプリ

HTML / CSS / JavaScriptで作成。

### できること
- 入力欄にToDoを入力できる
- 追加ボタンでリストに表示できる
- 空欄のまま追加できない
- ❌ボタンで削除できる
