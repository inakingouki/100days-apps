# 100days-apps

AIを活用しながら100日間アプリ開発に挑戦するリポジトリです。

## Day1: じゃんけんゲーム

HTML / CSS / JavaScriptで作成。

### できること
- グー・チョキ・パーを選べる
- CPUがランダムに手を出す
- あいこ・勝ちの判定

## Day2: 電卓アプリ

HTML / CSS / JavaScriptで作成。

### できること
- 数字ボタンを押すと表示される
- 四則演算ができる
- Cボタンでリセットできる

## Day3: ToDoアプリ

HTML / CSS / JavaScriptで作成。

### できること
- 入力欄にToDoを入力できる
- 追加ボタンでリストに表示できる
- 空欄のまま追加できない
- ❌ボタンで削除できる


## Day4: タイマーアプリ

HTML / CSS / JavaScriptで作成。

### できること
- 25分タイマーを表示できる
- スタートでカウントダウンできる
- ストップで一時停止できる
- リセットで25:00に戻せる
- スタート連打で速度が上がるバグを防止
- 0秒になったら停止して通知する

## Day5: 文字数カウンター

HTML / CSS / JavaScriptで作成。

### できること
- 入力した文章の文字数をリアルタイムで表示できる
- 空白なしの文字数を表示できる
- X投稿用に残り文字数を表示できる

### 学んだこと
- `addEventListener("input")` で入力中に処理を実行する
- `length` で文字数を数える
- `replace(/\s/g, "")` で空白を除外する

## Day7: 健康目安チェッカー

HTML / CSS / JavaScriptで作成。

### できること
- 身長・体重・目標体重を入力できる
- BMIを計算できる
- BMIに応じて判定を表示できる
- 標準体重を表示できる
- 標準体重との差を表示できる
- 目標体重との差を表示できる
- アドバイスを表示できる
- 入力値と体重記録を保存できる

### 学んだこと
- `toFixed(1)` で小数点第1位まで表示する
- `if / else if / else` で判定を分岐する
- `localStorage` でブラウザにデータを保存する
